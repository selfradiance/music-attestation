Music Attestation Manifest

This repo is a public wax seal for audio files. The audio stays wherever the artist keeps it. The manifest records a file hash, a signature, and a date so anyone can later check whether a copy matches the sealed version.

What This Proves

Each seal is made from the SHA-256 hash of an audio file. That hash is signed with a private Ed25519 key. The private key is not published. The public key below lets anyone verify that a manifest entry was signed by the matching private key and that the file they have produces the same hash.

Verify A File

Clone this repo, put the audio file anywhere on your machine, then run:

node verify.mjs /full/path/to/audio-file.wav

A matching file prints PASS and the sealed date. A changed file, missing manifest entry, or invalid signature prints FAIL.

The manifest attests to file bytes, not authorship law, copyright ownership, or audio quality. If one byte changes, the hash changes.
